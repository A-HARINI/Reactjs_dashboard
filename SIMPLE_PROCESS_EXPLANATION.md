## ğŸ›  Tools We Use (Simple Explanation)

### **1. React**
- **What it is:** Library for building user interfaces
- **What it does:** Helps us create components (reusable pieces)
- **Why we use it:** Makes building websites easier

**Simple analogy:** Like LEGO blocks - we build pieces and put them together

---

### **2. Vite**
- **What it is:** Build tool and development server
- **What it does:** 
  - Runs our code locally (development)
  - Compiles code for production (build)
- **Why we use it:** Very fast, makes development easier

**Simple analogy:** Like a translator - converts our code into something browsers understand

---

### **3. Recharts**
- **What it is:** Library for making charts
- **What it does:** Creates bar charts, pie charts, etc.
- **Why we use it:** Makes data visualization easy

**Simple analogy:** Like a chart-making tool - we give it data, it makes charts

---

### **4. Vercel**
- **What it is:** Platform for hosting websites
- **What it does:** Puts our website on the internet
- **Why we use it:** Free, easy, automatic deployment

**Simple analogy:** Like a web hosting service - stores our website and makes it accessible
